# Akira Cooper – IT Portfolio

Welcome to my IT Portfolio! This repository showcases hands-on projects and labs I completed while studying for the **CompTIA A+ (Core 1 & Core 2)** certification. It is designed to demonstrate my practical IT support, troubleshooting, and documentation skills for entry-level help desk and IT support roles.

---

## Portfolio Overview

### 1. Troubleshooting Labs
Folder: `IT-Portfolio/Troubleshooting-Labs`  
Projects that demonstrate problem-solving for common IT issues:

- **PC-Build-and-Troubleshooting** – Hardware installation, BIOS setup, boot failures, and component troubleshooting.  
- **Windows-Installation** – Installing Windows, configuring user accounts, drivers, updates, and resolving installation issues.  
- **Network-Troubleshooting** – Diagnosing network problems using `ipconfig`, `ping`, `tracert`, and `nslookup`.

[Explore Troubleshooting Labs →](IT-Portfolio/Troubleshooting-Labs)

---

### 2. Help Desk
Folder: `IT-Portfolio/Help-Desk`  
Simulated help desk tickets and IT knowledge base guides demonstrating end-user support:

- **Help-Desk-Ticket-Examples** – Simulated tickets covering printer issues, internet connectivity, locked accounts, and slow computers.  
- **IT-Knowledge-Base** – Step-by-step technical guides for resetting passwords, mapping drives, troubleshooting printers, and fixing WiFi issues.

[Explore Help Desk →](IT-Portfolio/Help-Desk)

---

### 3. Home Lab
Folder: `IT-Portfolio/Home-Lab`  
Virtual lab environment to practice systems administration:

- **Active-Directory-Lab** – Creating users, groups, and policies in Windows Server. Practicing account management and permissions in a simulated network.

[Explore Home Lab →](IT-Portfolio/Home-Lab)
